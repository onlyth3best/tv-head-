# tv-head

<br/> A project I've been wanting to complete for a while.
<br/> It's a TV head cosplay-adjascent hardware:
- running on a RaspberryPi5
- using a monitor display
- carboard for the shell
- the art is created in Aseprite!!
- stylised for a cosplay ;p

## Motivation:
<br/> Fallout is the first time I've started a hardware project, so I wanted to make something I'd use and something that might combine making art with the project itself. and, a TV head looks really awesome!!
